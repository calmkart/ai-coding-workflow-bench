# Task: 修复字符串反转的多字节字符处理

## 目标
修复 Reverse 函数：当前按 byte 反转导致多字节 UTF-8 字符（如中文、emoji）损坏，应按 rune 反转。

## 变更范围
- strkit.go: 修复 Reverse 函数实现

## 具体要求
- REQ-1: ASCII 字符串正确反转（如 "hello" -> "olleh"）
- REQ-2: 中文字符串正确反转（如 "你好" -> "好你"）
- REQ-3: emoji 字符串正确反转
- REQ-4: 空字符串返回空字符串

## 约束
- Reverse 函数签名不可更改
- 使用 rune 切片进行反转

## 测试策略
- 验证 ASCII 字符串反转
- 验证中文字符串反转
- 验证 emoji 字符串反转
- 验证空字符串

## 不做什么
- 不添加新函数
- 不修改其他函数
